# 📰 Fake News Detection using NLP & Machine Learning

## 🚀 Overview
This project is a Natural Language Processing (NLP)-based Fake News Detection system that classifies news articles as REAL or FAKE using machine learning techniques.
It uses TF-IDF vectorization for feature extraction.



## 🎯 Problem Statement
In the digital age, misinformation spreads rapidly through online platforms, making it difficult to distinguish between real and fake news. Manual verification is not scalable due to the large volume of data.
This project aims to build an automated system that can analyze news text and accurately classify it as real or fake, helping reduce the impact of misinformation.

## 🧠 Objective


* Build an NLP-based classifier for fake news detection


* Convert text data into numerical features using TF-IDF


* Train a machine learning model to classify news articles


* Deploy a simple web interface using Streamlit
